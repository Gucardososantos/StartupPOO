# StartupPOO

O StartupGame é um jogo de simulação onde o usuário gerencia uma startup, toma decisões estratégicas (Marketing, Produto, Equipe, Investidores, Cortar Custos) e acompanha o impacto financeiro, reputacional e motivacional ao longo de várias rodadas. A pontuação final e o ranking são calculados automaticamente, com persistência dos dados no banco H2.

#Tecnologia Utilizadas

Tecnologias Utilizadas,Java 17,Maven,Banco de dados H2,JUnit 5 (testes),POO avançado,Estratégia de padrões, Fábrica,Repositório,Configuração via game.properties

#Como compilador

git clone <URL_DO_REPOSITORIO> cd StartupGame

Instale as dependências e compile mvn clean install

mvn exec:java -Dexec.mainClass="Main"
